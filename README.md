# User Registration and Authentication System  
## Description
This project is a full-stack User Registration and Authentication System developed for IT342 – Systems Integration and Architecture.  

The system includes a Spring Boot backend and a ReactJS frontend that supports secure user registration, login, profile access, and logout functionality.

Passwords are securely encrypted using BCrypt, and protected endpoints are secured using Spring Security.
